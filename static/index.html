Security & Usage

Pick a target under runtime settings before starting a session. The agent will inspect that host through SSH tools instead of the local machine.

Default is read-only mode (logs, status, sysinfo). Enable "Read-write mode" to let the agent run changing commands; every remote command is written to ~/.ccb/remote_audit.log.

The target only needs an SSH server. Use key-based auth and a least-privilege account; restrict its sshd to this machine.
